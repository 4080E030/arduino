# 排阻arduino 
## 程式
### 一樣程式 只是arduino 接腳不一樣
```arduino
int a=9;
int b=10;
int c=11; 
void setup() {

   pinMode(a, OUTPUT);
   pinMode(b, OUTPUT); 
   pinMode(c, OUTPUT);
}


void loop() {
  digitalWrite(a, 1); 
  digitalWrite(b, 0); 
  digitalWrite(c, 0); 
  delay(1000);                       
  
  digitalWrite(a, 0); 
  digitalWrite(b, 1); 
  digitalWrite(c, 0); 
  delay(1000); 
  
  digitalWrite(a, 0); 
  digitalWrite(b, 0); 
  digitalWrite(c, 1); 
  delay(1000);          
}
```
